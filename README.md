# Операції з вказівниками на дані різних типів
**Мета:** написати, відлагодити та протестувати програму, у якій дослідити операції з вказівниками на дані різних типів.

## Вказівки до роботи
Відповідно до цієї інструкції вам потрібно у середовищі Visual Studio Code з використанням набору компіляторів GCC створити програму мовою C операції з вказівниками на дані різних типів.
1.	Ознайомитися із оголошенням масивів та їх використанням.
2.	Вибрати завдання згідно свого варіанту у ДОДАТКУ.
3.	Розробити блок-схему алгоритму програми.

## Хід роботи:
Рекомендації до роботи:
1.	Продумайте, які типи змінних вам потрібно використати для виконання завдання.
2.	Використання вказівників є обов'язковим для виконання ключових операцій, зазначених у завданні.
3.	Програма повинна містити коментарі, що пояснюють ключові етапи роботи з вказівниками.
4.	Програма повинна виводити результати в консоль у читабельному форматі, з поясненнями, що саме виводиться (наприклад, "Початкове значення int:", "Адреса третього елемента:").
5.	Для варіантів, де це вимагається, надайте коротке текстове пояснення різниці у зміщенні адрес або розміру зміщення.

## Оформлення звіту
У звіті студент має вказати свій варіант індивідуального завдання та представити:
1.	Блок-схему алгоритму програми.
2.	Код (лістинг) програми.
3.	Результати тестування для різних вхідних умов.
4.	Конструктивний висновок — що ви дізналися нового, чого навчилися, якими способами та інструментами досягнули мети.

## ДОДАТОК
### Варіанти завдань

#### Варіант 1
Оголосіть змінні `int`, `float` та `double`. Створіть вказівники на них. Використовуючи оператор розіменування `*`, збільште значення кожної змінної на 10 (для `int`), 5.5 (для `float`) та 20.25 (для `double`). Виведіть початкові та кінцеві значення.

#### Варіант 2
Оголосіть змінні `short` та `long`. Створіть вказівники на них. Здійсніть приведення типу вказівника на `short` до вказівника на `void` (`void*`). Виведіть адреси змінних, а потім виведіть значення змінних, використовуючи відповідні вказівники та оператор розіменування.

#### Варіант 3
Оголосіть дві змінні: `char c = 'A'` та `int i = 100`. Створіть вказівники на них (`char* pc, int* pi`). Виведіть адреси, на які вони вказують. Додайте до `pc` 3 та до `pi` 5. Виведіть нові адреси. Поясніть різницю у зміщенні адрес.

#### Варіант 4
Оголосіть змінні `int a = 5`, `int b = 15`. Створіть два вказівники `int* ptr_a` та `int* ptr_b`, які вказують на `a` та `b` відповідно. Напишіть код для обміну значень змінних `a` та `b`, використовуючи лише вказівники та оператор розіменування (без використання імен змінних `a` та `b` після їх оголошення).

#### Варіант 5
Оголосіть змінну double `d = 3.14159`. Створіть вказівник на неї. Використовуйте оператор розіменування для того, щоб вивести значення змінної, а потім, додавши 1 до вказівника, виведіть нову адресу. Поясніть, на скільки байтів відбулося зміщення.

#### Варіант 6
Оголосіть змінні `char c`, `int i`, `float f`. Створіть вказівник на `void` (`void*`) і по черзі присвоюйте йому адреси `c`, `i` та `f`. Після кожного присвоєння виведіть значення змінної, використовуючи приведення типу для розіменування `void*`.

#### Варіант 7
Оголосіть змінну `char c = 'Y'`. Створіть вказівник `char* pc`. Здійсніть арифметичну операцію додавання: `pc + 10`. Виведіть адресу `pc` та нову адресу. Виведіть різницю між цими адресами в байтах та поясніть результат.

#### Варіант 8
Оголосіть змінну `int x = 42`. Створіть подвійний вказівник `int** ptr_ptr` та ініціалізуйте його, щоб він вказував на вказівник, який вказує на `x`. Виведіть адресу `x`, адресу вказівника, який вказує на `x`, та значення `x`, використовуючи лише `ptr_ptr` і відповідні оператори розіменування.
